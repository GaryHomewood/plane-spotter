# Planespotters

An Android app to retrieve and display an RSS feed.

## Features

1. MVP architecture
2. Retrofit 2 with caching
3. RxJava
4. Glide
5. RSS feed modelled with [Parceler](https://github.com/johncarl81/parceler)
6. Realm.io used to save favourites
